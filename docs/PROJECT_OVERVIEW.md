# 🧠 Mitra 2.0 — Project Overview

> **"Mitra"** means **friend** — because every student deserves a trusted companion for their mental well-being.

---

## 🎯 Vision

Build a **digital mental health support platform** that empowers college students to:

- Detect mental health issues **early**
- Access **anonymous, stigma-free** support
- Get **AI-guided coping strategies** in real-time
- Connect with **counsellors** effortlessly
- Provide **data insights** to institutions for better student care

---

## 🌍 Problem Statement

| Statistic                                             | Impact                             |
| ----------------------------------------------------- | ---------------------------------- |
| **70%** of college students report significant stress | Academic performance drops         |
| **40%** experience sleep problems                     | Physical and mental health decline |
| **25%** have moderate-to-severe anxiety               | Daily functioning impaired         |
| **Less than 30%** seek professional help              | Fear of stigma prevents access     |
| **Limited counsellor availability**                   | 1 counsellor per 1000+ students    |

### Why Mitra 2.0?

- **Stigma barrier** → Anonymous support removes fear
- **Access barrier** → 24/7 AI chatbot as first responder
- **Awareness gap** → Validated screening tools detect issues early
- **Resource gap** → Psychoeducational content empowers self-care
- **Institutional blindspot** → Analytics dashboard reveals campus-wide trends

---

## 👥 Target Users

### 1. 🎓 Students (Primary Users)

- Take mental health assessments
- Chat with AI assistant
- Book counsellor appointments
- Access educational resources
- Participate in peer support forums
- Track daily mood

### 2. 🤝 Peer Volunteers

- Provide peer-to-peer support in forums
- Trained in basic mental health first aid
- Moderate community discussions
- Escalate serious cases

### 3. 🧑‍⚕️ Counsellors

- Manage appointment schedules
- View student assessment history (with consent)
- Conduct online sessions
- Receive AI-flagged high-risk cases

### 4. 🏛️ College Administration

- View anonymized analytics dashboard
- Identify campus-wide mental health trends
- Plan interventions (workshops, extra counsellors)
- Monitor system usage metrics

## 🏆 Hackathon Differentiators

What makes Mitra 2.0 **stand out**:

1. **AI-Powered Risk Detection** — Not just a chatbot, but an emotion-aware system that detects crisis signals
2. **Validated Clinical Tools** — Uses real psychological instruments (PHQ-9, GAD-7)
3. **Privacy-First Design** — Anonymous access, encrypted data, no judgment
4. **Institutional Impact** — Aggregated insights help colleges take action
5. **Multi-Language Support** — English, Hindi, Odia, and regional languages
6. **Peer-to-Peer Layer** — Students helping students, moderated by AI
